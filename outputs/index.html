    Evidências do Passo a Passo

    1. Verificando a Instalação do Terraform

    Após instalar o Terraform, você pode verificar a instalação executando o comando terraform -v.

    Comando:

    terraform -v

    Descrição do Print: Este print mostra a versão do Terraform instalada no seu sistema, confirmando que a instalação foi bem-sucedida.

    2. Verificando a Instalação do AWS CLI

    Depois de instalar o AWS CLI, você pode verificar a instalação executando o comando aws --version.

    Comando:

    aws --version

    Descrição do Print: Este print mostra a versão do AWS CLI instalada, garantindo que a instalação foi concluída corretamente.

    3. Criando o Arquivo main.tf

    Crie um arquivo de configuração Terraform chamado main.tf com o seguinte conteúdo:

    provider "aws" {
  region = "us-east-1"
}

resource "aws_instance" "example" {
  ami           = "ami-0c02fb55956c7d316"  # Amazon Linux 2 AMI (HVM), SSD Volume Type
  instance_type = "t2.micro"

  tags = {
    Name = "TerraformExample"
  }
}

    Descrição do Print: Este print mostra o conteúdo do arquivo main.tf criado no seu editor de texto.

    4. Inicializando o Terraform

    Inicialize o Terraform executando o comando terraform init no diretório onde está o arquivo main.tf.

    Comando:

    terraform init

    Descrição do Print: Este print mostra a saída do comando terraform init, indicando que o Terraform foi inicializado com sucesso.

    5. Planejando a Infraestrutura

    Execute o comando terraform plan para ver o que será criado.

    Comando:

    terraform plan

    Descrição do Print: Este print mostra a saída do comando terraform plan, detalhando os recursos que serão criados.

    6. Aplicando a Configuração

    Para criar a instância EC2, execute o comando terraform apply e digite yes quando solicitado.

    Comando:

    terraform apply

    Descrição do Print: Este print mostra a saída do comando terraform apply, indicando a criação bem-sucedida da instância EC2.

    7. Verificando a Instância na AWS

    Para verificar se a instância foi criada, acesse o AWS Management Console, navegue até o serviço EC2 e confira a lista de instâncias.

    Descrição do Print: Este print mostra a instância EC2 criada na AWS. Ele inclui detalhes como o ID da instância, tipo de instância, estado e nome ("TerraformExample").

    8. Limpando a Infraestrutura

    Para destruir a instância criada, execute o comando terraform destroy e digite yes quando solicitado.

    Comando:

    terraform destroy

    Descrição do Print: Este print mostra a saída do comando terraform destroy, indicando a destruição bem-sucedida da instância EC2.
